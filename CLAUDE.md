# Description

A web application that allows users to manage their Magic: The Gathering collection.

# Initial Features

- Complete Magic: The Gathering card database
- Account creation/authentication
- Search cards by name
- Create collections
- Add cards to collections
- Share collections with users (i.e., a couple can have access to their shared collection on MagicBox)

# Technologies

- Laravel

# Coding Style Guide

- Prioritize Laravel code, don't use hacky PHP workarounds unless proposed and accepted by me
- Do not use emojis
